## 如何能作出和这个Github repository一样的工程
### HTML的基本结构
```html
<html>
    <head>
    </head>
    <body>
    </body>
</html>
```

#### 指明页面的title
```html
<head>
    <title>页面Title</title>
</head>
```

#### 在页面中引入css文件
```html
<link rel="stylesheet" href="./mycss.css" />
```

#### 使用ul
```html
<ul>
    <li>
        Github地址
    </li>
    <li>
        博客地址
    </li>
</ul>
```

#### header作为页面的header或者文章的header
```html
<header>
    <h1>
        页面的头
    </h1>
</header>

```

```html
<article>
    <header>
        <h1> 背影</h1>
    </header>
    <p>你在这别动，我去买几个橘子</p>
</article>
```

### 使用img引入图片
